# 🗒 Description

Vue SQL Editor.
<br />
Programing with `VueJS 2.6`.

<br />

# ✨ Features

1. SQL highlight
2. SQL format
3. SQL prompt

<br />

# 🔨 Installation

```
npm install vue-sql-editor --save
or
yarn add vue-sql-editor
```

<br />

# 🏗 Usage

```javascript
import SQLEditor from 'vue-sql-editor'

<SQLEditor v-model="sqlstr" :readOnly="false" />
```

or

```javascript
<sql-editor v-model="sqlstr" :readOnly="false" />
```
